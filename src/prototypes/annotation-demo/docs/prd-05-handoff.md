# 交付

## 1. 交付物

- 可运行原型：包含原型即 PRD、内容标注、状态标注、原型目录、开启标注、编辑标注和 Agent 读取七个页面。
- 标注数据：包含节点定位、颜色分类、Markdown 正文和状态控制配置。
- PRD 文档：包含总览、角色、流程、状态、风险和交付六个章节化文档。
- 外部来源：只保留必要的外部设计来源链接，避免目录成为无关资料集合。

![文档编辑](__ANNOTATION_IMAGE_DOCUMENT_EDIT__)

## 2. 验收标准

- 目录中页面入口必须能切换到对应页面。
- 内容标注支持 Markdown、多颜色分类、颜色筛选和同一节点多条标注。
- 状态标注能通过分段控件切换成功/失败、空列表/有内容、偏低/正常/偏高。
- 开启标注、编辑标注和 Agent 读取三章应保持说明克制，不重复展开。

## 3. 后续扩展

后续可以把这套目录结构作为项目模板：新原型先建立页面和状态，再开启标注、编辑标注，并让开发 Agent 读取完整上下文。
