# 添加厂商表单

## 怎么进入

在列表上方第二行点击【添加厂商】，右侧弹出抽屉，标题为「添加厂商」，底部为【取消】和【确定】。

## 表单分两块

### 厂商信息

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 厂商编号 | 否（只读） | 添加时固定显示「-」，保存后由系统分配 |
| 厂商名称 | 必填 | 签约使用的公司全称 |
| 联系人 | 选填 | |
| 手机 | 选填 | |
| 邮箱 | 选填 | |
| 单位地址 | 选填 | |
| 发票信息 | 必填 | 下拉选择：专票 6%、专票 3%、专票 1%、普通发票、其他 |

### 银行信息

以下五项均为必填：开户名称、开户银行、开户银行所在地、支行名称、银行卡号。其中开户银行、开户银行所在地为下拉选择，其余为文本输入。

## 保存时的规则

共有七项必填：厂商名称、发票信息，以及银行信息五项。联系人、手机、邮箱、单位地址可以不填。

| 情况 | 页面表现 |
|------|----------|
| 必填未齐 | 未填字段下方红色提示，并弹出「请完善所有信息」 |
| 保存成功 | 关闭抽屉，列表新增该厂商，并弹出绿色「提交成功」 |

新厂商编号按当前业务类型在对应号段内递增：游戏盒一般从 1001 起，快爆一般从 2001 起。
