# 发票信息列

## 做什么

点击表头旁的筛选图标，可按发票类型过滤列表中的厂商。

## 可选项

- 增值税专用发票（6%）
- 增值税专用发票（3%）
- 增值税专用发票（1%）
- 普通发票
- 其他

## 不同情况怎么展示

| 情况 | 列表表现 |
|------|----------|
| 未选任何筛选 | 显示所有发票类型 |
| 选中某一类型 | 只显示该类型的厂商 |
| 单元格内容 | 直接显示发票类型文字，不用彩色标签 |

## 和其他功能的关系

在「结算公式管理」中，若扣税方式选「跟随发票」，会按这里的发票类型自动带出对应税率；若发票类型为「其他」，则需在公式里手动填写扣税点。
