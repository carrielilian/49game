# 编辑厂商表单

## 表单结构

与添加表单布局相同，分为「厂商信息」和「银行信息」两块，内容来自当前正在编辑的那一行。

### 厂商信息

| 字段 | 是否必填 | 编辑时怎么展示 |
|------|----------|----------------|
| 厂商编号 | 否（只读） | 显示当前编号，不可修改 |
| 厂商名称 | 必填 | 可改，默认是列表里的名称 |
| 联系人 | 选填 | 可改 |
| 手机 | 选填 | 可改 |
| 邮箱 | 选填 | 可改 |
| 单位地址 | 选填 | 可改 |
| 发票信息 | 必填 | 下拉，默认是列表里的类型；若修改，可能影响该厂商相关结算公式中的扣税点展示 |

### 银行信息

开户名称、开户银行、开户银行所在地、支行名称、银行卡号均为必填，均可编辑；银行与所在地为下拉，其余为输入框。

## 保存规则

七项必填：厂商名称、发票信息，以及银行信息五项。联系人、手机、邮箱、单位地址可以不填。

| 情况 | 页面表现 |
|------|----------|
| 必填项为空 | 该字段下方显示「某某不能为空」类红色提示 |
| 点保存但未通过校验 | 抽屉保持打开，并弹出「请完善所有信息」 |
| 保存成功 | 关闭抽屉，列表该行更新，厂商编号不变，并弹出绿色「提交成功」 |

## 业务说明

- 编辑不能改厂商编号，也不能把厂商挪到另一个业务类型（游戏盒 / 快爆）。
- 若改了发票信息，该厂商旗下游戏在「结算公式管理」里选用「跟随发票」扣税方式时，展示的税率会随新发票类型变化；若发票选「其他」，则需手动填写扣税点。
