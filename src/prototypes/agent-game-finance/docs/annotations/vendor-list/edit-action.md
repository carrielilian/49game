# 编辑

## 做什么

点击某一行的【编辑】，从右侧打开「编辑厂商」抽屉，自动带入该行已有信息。厂商编号不可修改。

## 和「添加厂商」的区别

| 对比项 | 添加 | 编辑 |
|--------|------|------|
| 抽屉标题 | 添加厂商 | 编辑厂商 |
| 底部主按钮 | 确定 | 保存 |
| 厂商编号 | 只读，显示「-」 | 只读，显示当前行编号（如 1001） |
| 保存结果 | 新增一条厂商 | 更新当前这一条 |

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 刚打开抽屉 | 表单已填好当前行数据，厂商编号只读 |
| 保存成功 | 关闭抽屉，列表中该行信息更新，并弹出绿色「提交成功」 |
| 必填未填 | 字段下方红色提示，并弹出「请完善所有信息」 |

各字段说明见 **编辑表单** 批注（需先打开编辑抽屉）。
