# 添加厂商

## 做什么

点击后从右侧滑出较宽的表单抽屉，用于录入新厂商信息。保存成功后关闭抽屉，新厂商会出现在当前业务类型下的列表中，系统会自动分配厂商编号。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 信息填写完整且合法 | 关闭抽屉，列表中出现新一行，并弹出绿色「提交成功」 |
| 有必填项未填 | 对应字段下方红色提示，并弹出「请完善所有信息」 |
| 添加过程中看厂商编号 | 表单里厂商编号为只读，显示「-」，保存后才由系统分配 |

## 表单详情

打开抽屉后，可查看 **添加表单** 批注，了解各字段含义与填写要求。
