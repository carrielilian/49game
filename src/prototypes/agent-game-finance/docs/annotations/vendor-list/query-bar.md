# 查询栏

## 做什么

用于缩小列表里要看的厂商范围，包含业务类型选择和两个搜索框。

- **业务类型**（最左侧下拉）：在「游戏盒」和「快爆」两套数据之间切换，列表只显示当前选中的那一套。
- **厂商编号、厂商名称**：两个独立搜索框，与业务类型同时生效（都要满足才显示）；支持模糊匹配。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 首次进入 | 默认业务类型为「游戏盒」，厂商编号一般为 1001 及以上 |
| 切换到快爆 | 只显示快爆下的厂商，编号一般为 2001 及以上 |
| 同时填多个条件 | 业务类型、编号、名称一起过滤 |
| 没有符合条件的数据 | 表格显示空状态，无数据行 |

## 数据范围

游戏盒与快爆的数据彼此独立，切换业务类型时不会混在一起显示。
