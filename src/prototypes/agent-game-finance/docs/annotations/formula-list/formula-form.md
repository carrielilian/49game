# 结算公式表单

## 顶部只读

游戏编号 / 游戏名称、厂商编号、厂商名称、当前已生效结算公式（两行；未配置时为「-」）。

## 内部 / 外部渠道结算公式设置

各区块包含：

| 字段 | 必填 | 说明 |
|------|------|------|
| 扣税点 | 必填 | 单选「跟随发票」或「自定义」 |
| 渠道费 | 必填 | 0–100 整数，带 % 后缀 |
| 分成 | 必填 | 0–100 整数，带 % 后缀 |

### 扣税点规则

| 模式 | 表现 |
|------|------|
| 跟随发票 | 按归属厂商「发票信息」映射为只读扣税点；发票为「其他」时需手输 |
| 自定义 | 手输百分数（如 6 表示 6%） |

| 发票信息 | 扣税点 |
|----------|--------|
| 增值税专用发票（6%） | 0% |
| 增值税专用发票（3%） | 3.36% |
| 增值税专用发票（1%） | 5.6% |
| 普通发票 | 6.72% |
| 其他 | 手输 |

## 保存规则

任一必填未填 → 红色提示 + 「请完善所有信息」。  
保存成功 → 关闭抽屉 + 绿色「提交成功」；列表公式列与操作记录同步更新。
