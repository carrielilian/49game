# 内部收入列表

## 做什么

展示内部渠道待结算与已结算记录。需先「数据拉取」，再点「结算」完成结算。

## 各列含义

| 列 | 说明 |
|----|------|
| 收入时间 | 拉取数据的月份（YYYY-MM） |
| 游戏ID / 游戏名称 | 左边游戏 ID，右边游戏名称 |
| 厂商编号、厂商名称 | 归属厂商，分两列 |
| 渠道 | 见 **渠道列** 批注 |
| 待结算金额 | 人民币 + 千分位 |
| 结算收入 | 未结算显示「-」；已结算显示人民币 + 千分位 |
| 结算公式 | 去掉前缀后的公式表达式 |
| 结算时间 | 未结算「-」；已结算为点击「结算」的时间 |
| 申请付款状态 | 见 **申请付款状态列** 批注 |
