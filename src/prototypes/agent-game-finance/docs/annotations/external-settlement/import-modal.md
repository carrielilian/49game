# 导入并结算弹窗

## 阶段一：上传

### 外部渠道类型

单选五选一：纯游外放、游乐外放、游乐IOS、快爆游IOS、49外放。须先选择才能上传报表。

### 上传报表

报表字段：**收入时间、游戏名称、厂商名称、待结算金额**。点击上传区模拟解析。

| 情况 | 表现 |
|------|------|
| 未选渠道就上传 | 红色「请先选择外部渠道类型」 |

## 阶段二：预览与结算

上传完成后切换为预览表格，顶部显示「外部渠道：{所选渠道}」。

| 列 | 说明 |
|----|------|
| 收入时间 | 来自报表 |
| 游戏编号 / 游戏名称 | 按报表「游戏名称」+「厂商名称」匹配系统游戏 |
| 厂商名称 | 来自报表 |
| 待结算金额 | 人民币 + 千分位 |
| 结算公式 | 匹配成功后读取该游戏外部渠道公式 |
| 结算收入 | 点击「结算」前为「-」；结算后显示金额 |

### 匹配与校验

| 情况 | 结算收入列表现 |
|------|----------------|
| 游戏名称或厂商名称无法匹配 | 红色「未匹配到游戏，请检查游戏名称与厂商名称」 |
| 游戏未配置结算公式 | 红色「该游戏尚未配置结算公式」 |

## 底部按钮

| 按钮 | 启用条件 |
|------|----------|
| 取消 | 始终 |
| 结算 | 已上传；全部结算成功后禁用 |
| 确认导入 | 全部行已匹配、已完成弹窗内结算且无错误；未满足时按钮置灰不可点 |

| 情况 | 表现 |
|------|------|
| 确认导入成功 | 关闭弹窗，主列表置顶新增记录，绿色「导入成功，已加入外部收入结算列表」 |
