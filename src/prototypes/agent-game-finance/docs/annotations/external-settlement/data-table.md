# 外部收入列表

## 做什么

展示已导入的外部渠道结算记录。本页数据均为**已结算**状态。

## 各列含义

| 列 | 说明 |
|----|------|
| 收入时间 | 报表中的收入月份 |
| 游戏ID / 游戏名称 | 左边游戏 ID，右边游戏名称 |
| 厂商编号、厂商名称 | 归属厂商，分两列 |
| 渠道 | 见 **渠道列** 批注 |
| 待结算金额 | 人民币 + 千分位 |
| 结算收入 | 已结算金额；人民币 + 千分位 |
| 结算公式 | 去掉前缀后的外部渠道公式表达式 |
| 结算时间 | 确认导入的时间，无表头筛选 |
| 申请付款状态 | 见 **申请付款状态列** 批注 |

## 业务说明

外部渠道不在主列表点「结算」，而是在「导入并结算」弹窗内完成结算后再导入。因此本列表不应出现结算收入或结算时间为「-」的行。
