# 查询栏

## 做什么

缩小主列表显示范围。

- **业务类型**：在「游戏盒」与「快爆」两套数据间切换。
- **退款时间**：按月范围筛选，默认近两个月（上个月 + 当前月）。
- **游戏编号 / 游戏名称、厂商编号、厂商名称**：分别匹配，与上述条件同时生效。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 首次进入 | 默认业务类型「游戏盒」，退款时间为近两个月 |
| 数据拉取成功后 | 时间范围保持近两个月，便于看到上月拉取的数据 |
| 无匹配数据 | 表格空状态 |

内部退款需先「数据拉取」，再在主列表点「结算」完成结算。与「内部收入结算」页的按钮状态、数据互不影响。
