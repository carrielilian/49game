# 数据拉取

## 做什么

根据渠道游戏 ID 去财务中心的对应渠道拉取待结算退款数据，写入主列表；4399 退款数据由开发者前台确认，故「数据拉取」仅拉取上次拉取时间到本次拉取时间期间在「财务中心-游戏退款结算」相匹配的数据。

各渠道和财务中心对应的业务类型如下：

| 渠道 | 财务中心对应数据类型 |
|------|----------------------|
| 快爆付费 | 好游快爆-手游&小游戏-手游付费下载 |
| 快爆内购 | 好游快爆-手游&小游戏-手游充值 |
| 快爆小游戏广告 | 好游快爆-小游戏广告-小游戏广告 |
| 游戏盒付费 | 四三九九-手游&微端-手游付费下载 |
| 游戏盒内购 | 四三九九-手游&微端-手游充值 |
| 49 广告联盟 | 四三九九-广告联盟-手游广告 |

## 规则

| 情况 | 页面表现 |
|------|----------|
| 每月首次进入 | 按钮可点，「结算」按钮置灰 |
| 财务中心的退款结算列表，快爆平台下存在游戏业务类型还没点击结算按钮 | 红色「财务中心还未结算完成」 |
| 拉取成功 | 绿色「已从财务中心拉取待结算数据」；「结算」变为可点；本页「数据拉取」置灰 |
| 同月再次进入 | 拉取成功后本页「数据拉取」保持置灰 |

## 数据来源

按各游戏结算公式「支持渠道」中已勾选的**内部渠道**及对应渠道游戏编号，从财务中心取数。拉取记录的「退款时间」为上月（YYYY-MM）。
