# 字段说明

## 做什么

面包屑标题右侧 **?** 图标，点击弹出各金额字段计算口径。

## 说明条目

1. 账户总收入 = 累计收入 − 累计退款  
2. 账户余额 = 内部与外部收入结算「未申请」结算收入之和 − 内部退款结算「未申请」结算退款之和（按游戏）  
3. 预付分成款 = 本页【付款设置】维护的预付分成款  
4. 已抵扣分成款 = 已付款待付款金额之和 + 历史已抵扣分成款；若预付分成款 − 上述合计 ≤ 0，则取预付分成款  
5. 剩余未抵扣分成款 = 预付分成款 − 已抵扣分成款（≤ 0 时为 0）  
6. 累计收入 = 内部与外部收入结算全部已结算「结算收入」之和  
7. 累计退款 = 内部退款结算全部已结算「结算退款」之和
