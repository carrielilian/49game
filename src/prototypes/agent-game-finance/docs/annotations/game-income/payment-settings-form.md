# 付款设置抽屉

## 区块一：预付分成管理

| 字段 | 规则 |
|------|------|
| 游戏ID / 游戏名称 | 只读；左边游戏 ID，右边游戏名称；打开抽屉时展示当前游戏 |
| 预付分成款 * | 必填；≥ 0；精确至小数点后两位；币种前缀取合同支付币种。**初始**：未保存过为空；已保存过显示已保存金额 |
| 历史已抵扣分成款 * | 必填；精确至小数点后两位；下方提示「请填写线下已抵扣处理的预付分成款，若无则填写0」。**初始**：未保存过为空（不默认 0.00）；已保存过显示已保存金额 |
| 已抵扣分成款 | 只读；不可编辑；编辑上方字段时不随输入变化。**初始**：预付分成款未填时「-」；已保存过显示按已保存数据计算的结果 |
| 剩余未抵扣分成款 | 只读；不可编辑；编辑上方字段时不随输入变化。**初始**：预付分成款未填时「-」；已保存过显示按已保存数据计算的结果 |

## 区块二：付费设置

| 字段 | 规则 |
|------|------|
| 分成付款公司 * | 4399 / 纯游 / 纯游（美元） / 香港4399 / 游家时代。**初始**：未保存过为空（显示「请选择」）；已保存过显示已选项 |
| 付款币种 * | 人民币 / 美金 单选。**初始**：未保存过默认选中「人民币」；已保存过显示已选项 |
| 付款账号 * | 文本输入。**初始**：未保存过为空；已保存过显示已保存内容 |

## 按钮

| 按钮 | 行为 |
|------|------|
| 取消 | 关闭，不保存 |
| 保存 | 校验通过后关闭；绿色「提交成功」 |

校验失败：字段下红字 + 红 Toast「请完善所有信息」。
