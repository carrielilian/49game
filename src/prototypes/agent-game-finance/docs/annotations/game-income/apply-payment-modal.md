# 申请付款确认

## 做什么

二次确认本次申请付款金额。

## 文案

| 情况 | 弹窗内容 |
|------|----------|
| 上月内部收入与内部退款均未完成结算 | 第一行：{上月}内部渠道还未结算，是否继续申请付款？；第二行：申请付款金额：{余额}元 |
| 两侧内部结算均已完成 | 仅一行：申请付款金额：{余额}元 |

## 按钮

| 按钮 | 行为 |
|------|------|
| 取消 | 关闭 |
| 确认申请 | 见下方成功逻辑 |

## 确认申请成功

1. 绿色提示「申请付款成功，账户余额已清零」
2. 按申请时刻信息固定保存本笔付款与结算函（详见「申请付款」批注）
3. 在「游戏付款管理」主列表**最上方**新增「未付款」记录
4. 本次纳入的结算明细改为「已申请」

失败（余额 ≤ 0）：红色提示「账户余额不足，无法申请」。
