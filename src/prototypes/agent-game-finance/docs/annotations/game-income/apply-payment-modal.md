# 申请付款确认

## 做什么

二次确认本次申请付款金额。

## 文案

| 情况 | 弹窗内容 |
|------|----------|
| 上月内部收入与内部退款均未完成结算 | 第一行：{上月}内部渠道还未结算，是否继续申请付款？；第二行：申请付款金额：{余额}元 |
| 两侧内部结算均已完成 | 仅一行：申请付款金额：{余额}元 |

## 按钮

| 按钮 | 行为 |
|------|------|
| 取消 | 关闭 |
| 确认申请 | 成功：绿色「申请付款成功，账户余额已清零」，并在「游戏付款管理」主列表**最上方**插入一条「未付款」记录；余额不足：红色「账户余额不足，无法申请」 |
