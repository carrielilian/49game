# 游戏收入列表

## 做什么

按游戏维度展示账户余额、累计收支与预付分成情况；余额大于 0 时可发起申请付款。

## 各列含义

| 列 | 说明 |
|----|------|
| 游戏ID / 游戏名称 | 左边游戏 ID，右边游戏名称 |
| 厂商名称 | 归属厂商名称（无厂商 ID 列） |
| 账户余额 | 见面包屑 **?** 字段说明 |
| 账户总收入 | 累计收入 − 累计退款；固定 ￥ |
| 预付分成款 | 未填显示「-」；已填按合同支付币种展示 |
| 已抵扣 / 剩余未抵扣分成款 | 预付未填时「-」；已填同合同支付币种 |
| 累计收入 / 累计退款 | 各结算页已结算记录之和；固定 ￥ |
| 操作 | 【付款设置】始终显示；余额 > 0 另显示【申请付款】 |
