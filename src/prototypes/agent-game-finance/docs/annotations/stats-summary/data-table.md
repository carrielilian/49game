# 收入汇总列表

## 做什么

按所选维度与时间范围，汇总展示支付金额、总收入与结算付款金额。

## 各维度列

| 维度 | 列 |
|------|-----|
| 游戏 | 时间、游戏ID / 游戏名称、支付金额、总收入、结算付款金额 |
| 渠道 | 时间、渠道、支付金额、总收入、结算付款金额 |
| 厂商 | 时间、厂商ID、厂商名称、支付金额、总收入、结算付款金额 |

## 金额含义

| 列 | 说明 |
|----|------|
| 支付金额 | 合同已付游戏代理金 + 已付预付分成款 + 已付委托开发费；按币种分别展示 |
| 总收入 | 结算收入 − 结算退款；固定 ￥ |
| 结算付款金额 | 游戏付款管理已付款记录的「实际付款金额」按维度与付款月份累加；固定 ￥ |

游戏ID / 游戏名称：左边游戏 ID，右边游戏名称。
