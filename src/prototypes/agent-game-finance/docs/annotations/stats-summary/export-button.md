# 导出

## 做什么

将当前查询结果导出为表格文件，便于线下核对或归档。

## 导出范围

- 导出**当前查询条件下**列表中的每一行汇总数据
- 字段与**当前维度**下列表列一致
- **不包含**「查询总计」行

## 导出文件字段

文件为表格格式（可用 Excel 打开），第一行为表头，之后每行对应一条汇总记录。

| 维度 | 列（顺序与列表一致） |
|------|----------------------|
| 游戏 | 时间、游戏ID / 游戏名称、支付金额、总收入、结算付款金额 |
| 渠道 | 时间、渠道、支付金额、总收入、结算付款金额 |
| 厂商 | 时间、厂商ID、厂商名称、支付金额、总收入、结算付款金额 |

金额写法与列表一致：支付金额按币种分别展示（多币种时单元格内换行）；总收入、结算付款金额为 ￥ 千分位格式。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 导出成功 | 弹出绿色「导出成功」 |
| 文件名 | 收入汇总统计-{维度}-{当天日期}，例如 `收入汇总统计-游戏-2026-07-20.csv` |
