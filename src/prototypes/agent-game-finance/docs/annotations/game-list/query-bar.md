# 查询栏

## 做什么

缩小列表显示范围，包含业务类型切换和多个搜索框。

- **业务类型**：在「游戏盒」与「快爆」两套游戏数据间切换。
- **游戏编号 / 游戏名称**：匹配上线后的正式游戏名或游戏编号。
- **合同游戏名称**：仅匹配签约合同里使用的游戏名（与列表「合同游戏名称」列对应）。
- **厂商编号、厂商名称**：分别匹配，与上述条件同时生效。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 首次进入 | 默认业务类型为「游戏盒」 |
| 多条件组合 | 所有已填条件同时过滤 |
| 无匹配数据 | 表格空状态 |
