# 编辑游戏表单

## 字段（按顺序）

| 字段 | 必填 | 编辑时 |
|------|------|--------|
| 游戏编号 | — | 只读 |
| 游戏名称 | 必填 | 可改 |
| 合同游戏名称 | 必填 | 可改 |
| 归属厂商 | — | 只读，显示「编号 / 名称」 |
| 付款方 | 必填 | 可改 |
| 游戏负责人 | 选填 | 可改 |
| 版号 | 必填 | 可改 |
| 运营状态 | 必填 | 可改；变更会记入操作记录 |
| 备注 | 选填 | 可改 |

## 保存规则

必填项未齐 → 红色提示 + 「请完善所有信息」。  
保存成功 → 关闭抽屉 + 绿色「提交成功」。归属厂商与游戏编号不变。
