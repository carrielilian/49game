# 导出

## 做什么

将当前查询结果导出为表格文件，便于线下核对或归档。

## 导出范围

- 导出**当前查询条件下**列表中的每一行游戏数据
- **不包含**「操作」列
- **不包含**列表底部的「总计」行

## 导出文件字段

文件为表格格式（可用 Excel 打开），第一行为表头，之后每行对应一条游戏记录。共 **10 列**，顺序如下：

| 列名 | 文件里写什么 |
|------|--------------|
| 付款方 | 与列表一致 |
| 游戏ID | 与列表一致 |
| 游戏名称 | 与列表一致 |
| 合同游戏名称 | 与列表一致 |
| 厂商ID | 与列表一致 |
| 厂商名称 | 与列表一致（按厂商 ID 取名称） |
| 已付游戏代理金 | 与列表一致：有值时写「币种符号 + 千分位金额」；无值或未勾选对应合作内容时写「-」 |
| 已付预付分成款 | 同上 |
| 已付委托开发费 | 同上 |
| 运营状态 | 与列表一致，例如「未上线」「已上线」 |

三项「已付」金额的数据来源与列表相同，均来自该游戏的合同管理。

## 不同情况怎么展示

| 情况 | 页面表现 |
|------|----------|
| 导出成功 | 弹出绿色「导出成功」 |
| 文件名 | 游戏管理-当天日期 |
