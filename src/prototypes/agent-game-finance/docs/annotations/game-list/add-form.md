# 添加游戏表单

## 字段（按顺序）

| 字段 | 必填 | 说明 |
|------|------|------|
| 游戏名称 | 必填 | 上线后正式名称；下方有说明文案 |
| 合同游戏名称 | 必填 | 签约合同用名；下方有说明文案 |
| 归属厂商 | 必填 | 可输入厂商ID或厂商名称搜索，下拉展示匹配结果 |
| 付款方 | 必填 | 下拉：4399、纯游、游乐、游戏之家、香港4399、游家时代 |
| 游戏负责人 | 选填 | |
| 版号 | 必填 | 单选：有 / 无 |
| 运营状态 | 必填 | 单选：未上线 / 已上线；默认未上线 |
| 备注 | 选填 | |

## 保存规则

必填项未齐 → 红色提示 + 「请完善所有信息」。  
保存成功 → 关闭抽屉 + 绿色「提交成功」。
