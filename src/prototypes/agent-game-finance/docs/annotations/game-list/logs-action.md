# 操作记录

## 做什么

打开只读抽屉，查看该游戏的历史操作日志，按操作时间从新到旧排列。

## 记录类型与展示

| 类型 | 何时写入 | 「操作」列展示 |
|------|----------|----------------|
| 添加游戏 | 新建游戏 | 固定文字「添加游戏」 |
| 运营状态变更 | 编辑游戏时运营状态改变 | 彩色状态标签（最新状态） |
| 合作状态变更 | 合同管理中合作状态改变 | 彩色状态标签（最新状态） |
| 合同变更 | 合同金额或三项已付金额改变 | 多行文字，格式为「字段名」变更为「新值」；金额带币种与千分位 |

合同变更具体比对哪些字段，见 **操作记录抽屉** 批注。

无记录时显示「暂无操作记录」。本抽屉仅查看，无提交按钮。
