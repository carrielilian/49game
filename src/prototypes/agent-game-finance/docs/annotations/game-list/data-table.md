# 游戏列表

## 做什么

展示当前业务类型与查询条件下的游戏清单，含付款、合同金额摘要及运营状态，可通过操作列进入各功能抽屉。

## 各列含义

| 列 | 说明 |
|----|------|
| 付款方 | 该游戏付款主体 |
| 游戏ID / 游戏名称 | 左边游戏 ID，右边游戏名称 |
| 合同游戏名称 | 合同签约用名，可与上线名不同 |
| 厂商编号、厂商名称 | 归属厂商，分两列 |
| 三项已付金额 | 见 **已付金额列** 批注 |
| 运营状态 | 见 **运营状态列** 批注 |
| 操作 | 编辑、合同管理、支持渠道、操作记录 |

## 排序

默认按添加时间从新到旧。
