# 合同管理表单

## 字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 游戏编号 / 游戏名称 | — | 只读 |
| 合同编号 | 必填 | |
| 支付币种 | 必填 | 人民币 / 美金；选定后下方金额输入框显示对应币种符号 |
| 合同金额 | 必填 | 两位小数，不小于 0 |
| 合作内容 | 必填 | 至少勾选一项：游戏代理金、预付分成款、委托开发费 |
| 已付游戏代理金 | 勾选代理金时必填 | 勾选后才显示 |
| 已付预付分成款 | 勾选预付分成款时必填 | 勾选后才显示 |
| 已付委托开发费 | 勾选委托开发费时必填 | 勾选后才显示 |
| 补充说明 | 选填 | |
| 合作状态 | 必填 | 合作中 / 合作终止；默认合作中 |

取消勾选某合作内容时，对应已付字段隐藏并清空。

## 保存规则

必填未齐 → 红色提示 + 「请完善所有信息」。  
保存成功 → 关闭抽屉 + 绿色「提交成功」；列表三项已付列与操作记录同步更新。

支付币种仅在本抽屉维护，影响列表金额展示币种。
